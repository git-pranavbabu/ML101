# ML101 - Machine Learning Basics

This directory contains introductory machine learning projects focusing on Linear Regression.

## Projects

- **Simple Linear Regression**: A basic implementation of linear regression to predict salary based on experience.
- **House Rent Prediction**: Predicting house rent based on features like size, bedrooms, and age.
- **Battery Life Prediction**: Predicting battery hours based on screen time, apps used, and network type.

## Datasets

- `simple_LR_dataset.csv`
- `house_rent.csv`
- `battery_life.csv`

## Requirements

To run these notebooks, you will need the following libraries:
- pandas
- matplotlib
- scikit-learn
- ipykernel (to run in Jupyter)

Install them using:
```bash
pip install -r requirements.txt
```
